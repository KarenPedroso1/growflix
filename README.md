O Growflix é uma plataforma web inspirada em serviços de streaming, desenvolvida com o objetivo de centralizar e apresentar conteúdos em vídeo da Growdev em um único ambiente digital.

O projeto foi construído utilizando HTML, CSS, JavaScript e Bootstrap, proporcionando uma interface moderna, responsiva e intuitiva para navegação. A aplicação organiza os conteúdos em diferentes categorias, permitindo que os usuários explorem facilmente episódios do Growcast, Webinars, Jornada UX/UI e outros conteúdos relacionados à tecnologia.
